# Race-Day Nutrition Calculator

Science-based nutrition calculator for endurance athletes by Keystone Endurance.

## Features

- Sprint, Olympic, Half, and Full Ironman triathlons
- 5K, 10K, Half Marathon, and Full Marathon runs
- Gender-specific calculations
- Imperial units (lbs, inches, oz)
- Science-based protocols with references
- 4-phase nutrition plan:
  - Taper nutrition (3 days before)
  - Race morning nutrition
  - During-race fueling
  - Post-race recovery

## Deployment to Vercel

1. Create a new repository on GitHub
2. Push this code to the repository
3. Connect repository to Vercel
4. Deploy!

Or use Vercel CLI:
```bash
npm install -g vercel
vercel
```

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view.
